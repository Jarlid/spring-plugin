Reports "Unknown reference (JSON)" when "$ref" value leads nowhere

Example:

"Unknown reference (JSON)" will be reported on "#/components/schemas/Unknown" if there is no such component Quick Fix adds it to the specification file 

    {
      "paths": {
        "/some/path": {
          "get": {
            "parameters": [
              {
                "$ref": "#/components/schemas/Unknown"
              }
            ]
          }
        }
      }
    }
